عون (Awn)

A shadcn-style utility library for TypeScript

Copy utilities into your project. No runtime dependencies.

npm install -g awn GitHub →
Installation Configuration Array Utils String Utils Number Utils

Installation

# Install globally
npm install -g awn

# Initialize configuration (optional)
npx awn init

# Add utilities to your project
npx awn add debounce

Configuration

Create a utils.json file to customize paths and aliases:

{
  "$schema": "https://awn.dev/schema.json",
  "utils": {
    "path": "./src/utils"
  },
  "aliases": {
    "utils": "~/utils",
    "~": "./src",
    "@": "./src"
  }
}
